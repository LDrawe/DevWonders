Clamp

You can use the clamp() function to define a range within which your element's size will adjust fluidly, accommodating various screen sizes and content lengths. It takes 3 values: minimum, preferred and maximum. So instead of using both min-width and max-width properties separately, you can achieve the same effect with a single clamp() function

PSA: Be sure to use a dynamic value for the preferred value

Resetting

If you ever needed to reset an element style and had to do for every property, you can just use the "all" shorthand

Gap

Achieving clean and consistent spacing between elements is a constant pursuit.This often involve meticulously setting margins on individual elements. Even justify-content will not always reproduce what you want.For this you can use the gap property. The gap property acts as a shorthand for row-gap and column-gap. It defines the size of the gap between the rows and between the columns in flexbox or grid layout No need to manually set margin for each element anymore!

Inset

Ever needed to manually set the top, bottom, right and left property of an element? This css property is a shorthand for that

Margin-inline

Speaking of inlines, you probably already familiar with margin by now, But what if you only want to apply the margin on the sides? You don't have to specify them anymore, just use the margin-inline property
