Big Numbers

If I ask you what's this number: 4.738, it's easy and quick to read. But what about this one: 1000000000 ? That probably took you a while to guess. You probably have seen a large number like this while coding. Thankfully, javascript has a way to make these more readable, you can put underlines 

                    
                        console.log(1_000_000_000)

Delete

If you have an object and need to delete a property from it, you may be tempted to use the delete keyword. And that's fine, but you can solve this in a more elegant and safer way by using destructuring.

Maior legibilidade: O destructuring torna o código mais claro e fácil de entender, pois explicita quais propriedades estão sendo removidas. Evita erros: A keyword delete pode ser perigosa se usada incorretamente, pois pode remover propriedades de objetos inesperadamente. O destructuring é mais seguro, pois apenas remove as propriedades especificadas.
